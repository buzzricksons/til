# フレームワーク
## Selenide
http://selenide.org/

### 特徴
- Javaで作られたSeleniumのラッパー
- IDEとの親和性が良い。(自動完成サポートなど)
- Ajaxなどの非同期処理に対する機能が良い
- Enterpriseサポートしている
- JUnit5対応

### Selenium vs Selenide
https://github.com/codeborne/selenide/wiki/Selenide-vs-Selenium

## Geb
http://gebish.org/
ジェブと呼ぶ。

### 特徴
- Groovyで作られたSeleniumのラッパー
- DSLを利用したjQuery感じのセレクターを提供
- wait処理が使いやすい
- Spockが利用可能
